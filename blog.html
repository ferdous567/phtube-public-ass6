1.Discuss the scope of var, let, and const...

Answer:

var: Variables declared with var are in the function scope.

let: Variables declared as let are in the block scope.

const: Variables declared as const are in the block scope.
2.Tell us the use cases of null and undefined.

Answer:

Undefined means the variable has been declared, but its value has not been assigned. Null means an empty value or a blank value.
3. What do you mean by REST API?

Answer:

An API, or application programming interface, is a set of rules that define how applications or devices can connect to and communicate with each other.
